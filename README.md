## Описание работы

В данной лабораторной работе реализована очередь на основе двух стеков с поддержкой ограничения максимального размера и проверкой на переполнение.

### Принцип работы

**Очередь (Queue)** работает по принципу FIFO (First In, First Out) — первый добавленный элемент удаляется первым.

**Стек (Stack)** работает по принципу LIFO (Last In, First Out) — последний добавленный элемент удаляется первым.

Чтобы реализовать очередь через два стека, используются два стека:

- **stack_input** — принимает новые элементы при добавлении в очередь (операция `enqueue`)
- **stack_output** — используется для удаления элементов из начала очереди (операция `dequeue`)

Когда необходимо удалить или просмотреть первый элемент, данные перекладываются из `stack_input` в `stack_output`. При этом порядок элементов меняется на обратный, и самый первый добавленный элемент оказывается на вершине второго стека. Благодаря этому извлечение происходит в правильном порядке FIFO.
